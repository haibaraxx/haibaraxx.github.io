《遥感原理与应用》课程笔记(一）：遥感的概念与遥感技术系统的组成

发表于 2017-10-07 | 分类于 Geoscience | 浏览 次

声明：本文参考课程《遥感原理与应用》，授课教师：方圣辉；参考教材《遥感原理与应用》，主编：孙家抦。

遥感的概念

广义：遥远的感知。
科学技术领域中：泛指一切无接触的远距离探测。
本课程讨论的遥感：使用安放在承载工具（平台）的某种装置（遥感器），在不直接接触被研究目标的情况下，感测目标的特征信息（一般是电磁波的反射辐射或者发射辐射），经过传输、处理，从中提取人们感兴趣的信息的过程。

遥感的类型

按遥感平台（的高度）分

  • 地面遥感：遥感器在地面上。
  • 航空遥感：遥感平台为飞机、飞艇、气球等。
  • 航天遥感：遥感平台以地球人造卫星为主。
  • 宇航遥感：对其他星球的遥感。

按传感器的探测波段范围分

  • 紫外遥感
  • 可见光遥感
  • 红外遥感：反映地面物体热辐射信息。
  • 微波遥感：地面物体很少反射和发射微波。是主动遥感的一种。

按工作方式分

  • 主动遥感：微波遥感是一种主动遥感。
  • 被动遥感：可见光遥感是一种被动遥感。

按记录信息的表现形式分

  • 成像遥感
  • 非成像遥感：不成像，而是记录距离等信息（如GPS）

按遥感的应用领域分

  • 外层空间遥感
  • 大气层遥感
  • 陆地遥感
  • 海洋遥感
  • 资源遥感
  • 农业遥感
  • 林业遥感
  • 地质遥感
  • 城市遥感
  • 军事遥感 …

遥感的特点

  • 真实性、客观性
  • 探测范围大
  • 资料新颖且能迅速反映动态变化
  • 成图迅速
  • 收集资料方便

遥感的过程

太阳发射的电磁波经过大气到达地球表面，经地物的反射、吸收、辐射后到达传感器，传感器收集数据（成像），并把数据传到地面站，地面站接收和预处理数据后，将其分发给用户，用户提取信息，并进一步分析与应用。

遥感技术系统组成

  • 目标的信息特性
  • 目标信息的传输
  • 空间信息采集
  • 地面接收与预处理
  • 信息处理
  • 信息分析与应用

遥感技术的发展

遥感平台

  • 不同高度、不同用途的卫星群
  • 多视角、多周期、全方位观测

遥感器

  • 高时间分辨力（单时相 -> 多时相，即看的时间间隔变小，次数增多）、高空间分辨力、高光谱分辨力（即在“颜色（谱）”上看得更细，或者说“颜色（谱）”分得更细，波段更短）
  • 探测波段范围不断延伸（如从可见光波段延伸到红外等其他波段）、多种技术集成等

遥感信息处理

  • 图像处理技术不断发展
    • 图像处理：光学处理 -> 数字处理（数据压缩、影像融合）
    • 信息提取：目视判读 -> 自动分类 -> 专家系统
    • 图像分析：定性 -> 定量
  • 图像处理软件不断完善
    • 软件：人机对话 -> 视窗式 -> 构件式、集成化、固件化
  • 与相邻学科的结合

遥感技术在中国

自主卫星

  • 海洋系列卫星：海洋1号、海洋2号、海洋3号
  • 气象系列卫星：风云1号（A\B）、风云2号、风云3号 (天气预报的卫星云图是从气象卫星获取的)
  • 资源系列卫星：资源1号（A\B）、资源2号（A\B）、资源3号
  • 环境系列卫星：环境1号（A\B\C）。1A是可见光的多光谱（4个波段）和高光谱卫星，1B是雷达卫星。
  • 等等。。。

遥感理论与应用

  • 利用多时相影像发现土地利用变化、农作物估产，精准农业中的应用
  • 气象、海洋、全球和局部环境监测与保护
  • 资源调查、自然灾害监测
  • 建设数字城市、数字省区、数字中国
  • 水利建设与规划，洪涝灾害监测
  • 医学、考古、质量检测
  • 军事应用：重要目标定位与侦查、导航与武器制导、打击效果评估、战场环境监测等。

遥感卫星发射基地

  • 酒泉卫星发射中心：中国目前唯一的载人航天发射场。
  • 西昌卫星发射中心：中国目前唯一发射地球同步卫星的航天发射场。
  • 太原卫星发射中心：发射了第一颗海洋资源勘查卫星。
  • 文昌卫星发射中心：距离赤道只有19°，地球自转造成的离心力可以让火箭负载更多的物品。

遥感教育

  • 中科院遥感应用研究所、地理研究所、对地观测中心
  • 武汉大学、北京师范大学、北京大学、长安大学、中南大学、浙江大学、北京航空航天大学等。

与其他学科的关系

  • 数字图像处理
  • 环境学、地理学
  • 计算机原理
  • 电磁波与电磁场
  • 测量学
  • 摄影测量学（航空遥感）
  • 地理信息系统
  • GPS原理与方法
